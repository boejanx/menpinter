# MENPINTER  
**Manajemen Penguatan Kompetensi Terpadu**

![MENPINTER](public/assets/img/logo/logo_kab.png)

MENPINTER adalah aplikasi manajemen penguatan kompetensi ASN yang terintegrasi untuk mendukung perencanaan, pelaksanaan, dan pemantauan pengembangan kompetensi secara efektif, transparan, dan berkelanjutan di lingkungan instansi pemerintah.
